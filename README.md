## Configuration

### Config file example (`./.vscode/applocalizer.json`)
```json
{
	"validator":

        "localizationFiles": "./locales/*.json"
    },
	"pseudo":
    {
        "expander": 30,
        "brackets": true,
        "accents": true
    }
}
```
